Initial prompt:
How can I commit my files to GitHub from PyCharm terminal?

Improved prompt:
CONTEXT: You will be provided with PyCharm GUI steps enclosed with {{{ STEPS }}} to execute a process enclosed with {{{ PROCESS }}}.
TASK: Convert the steps to CLI commands.

PROCESS: {{{ Commit and push files to a remote git branch }}}

STEPS: {{{
1. Review changed files
2. Stage the desired files
3. Add a commit message
4. Commit the files
5. Validate branch name
6. Push the changes to the remote branch
}}}

CLI COMMANDS:

# Step 1: Review changed files
git status

# Step 2: Stage the desired files (replace <file_name> with the actual file name or use '.' to stage all changes)
git add <file_name>  #: git add file1.py
# OR stage all files:
git add .

# Step : Add a commit message (replace <commit_message> with your message)
git commit -m "<commit_message>"  # Example: git commit -m "Added new feature"

# Step 4: Commit the files (already done in the previous step with git commit)

# Step 5: Validate branch name
git branch  # This will show the current branch name
# OR:
git rev-parse --abbrev-ref HEAD  # This will output the current branch name directly

# Step 6: Push the changes to the remote branch (replace <branch_name> with your branch name)
git push origin <branch_name>  # Example: git push origin
