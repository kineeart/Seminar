# Chuong 16 - Architecture, Design, and the Future

Bo tai lieu nay duoc soan de nop bai phan cong ca nhan cho Chuong 16.

## Thanh phan

- `report_ch16.md`: Bao cao day du theo huong hoc thuat + ung dung.
- `slides_ch16.md`: Dan y slide de dua vao PowerPoint/Google Slides.
- `code_demo/`: Code minh hoa y tuong kien truc va governance khi dua GenAI vao san xuat.
- `worklog_template.csv`: Mau nhat ky lam viec tung thanh vien de day len GitHub.

## Cach dung nhanh

1. Doc `report_ch16.md`, chinh sua ten, ma SV, lop, nhom.
2. Mo `slides_ch16.md`, copy tung slide vao PowerPoint.
3. Chay demo:

```bash
cd ch16/code_demo
python genai_architecture_decision.py
```

4. Dien `worklog_template.csv` va commit len GitHub theo tung ngay.

## Goi y xuat file nop

- Bao cao DOCX: copy noi dung `report_ch16.md` vao Word, chuan hoa heading, chen hinh mermaid ve lai bang draw.io/PowerPoint.
- Slide PPTX: dung `slides_ch16.md` lam script thuyet trinh.
- Code minh hoa: giu nguyen thu muc `code_demo/`.
